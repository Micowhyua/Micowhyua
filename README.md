## 💫 About Me
Saya adalah seorang Frontend Engineer yang berfokus pada pembangunan antarmuka web yang interaktif, responsif, dan berperforma tinggi. Saya senang mengubah desain UI/UX yang rumit dari Figma menjadi komponen kode yang *clean*, presisi (*pixel-perfect*), dan animatif.

- 🔭 Saat ini sedang aktif mengembangkan aplikasi web & mobile berbasis **Vue.js** dan **React/Next.js**.
- 🎨 Gemar mengeksplorasi animasi modern menggunakan **GSAP** dan styling super cepat dengan **TailwindCSS**.
- ⚡ Terus mendalami optimasi performa *frontend* (Web Performance) agar *loading* aplikasi menjadi super ringan dan responsif.
- ☁️ Memanfaatkan layanan cloud seperti **Google Cloud Platform** dan **Supabase** untuk mendukung kebutuhan deployment serta arsitektur serverless aplikasi frontend.
